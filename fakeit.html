SSO Integrations

Single Sign-On allows users using the Forethought Dashboard to be able to login through their own SSO provider without the need to have a separate username and password. You can find detailed instructions below the integration steps.

To reconfigure Okta, please disable the Okta integration and go through the steps to enable again.
Step 1: Choose your SSO provider
Step 2: Enter your SAML/IdP Metadata URL
Step 3: Click below to go to the final set of instructions
Learn how to bookmark the app
Setup Instructions
  1. In Okta Admin, select Applications then Add Application and search for “Forethought Dashboard”. Select Add.
    1. Select the Forethought app and go to the sign on tab.
    2. Right click the link for Identity Provider metadata and select Copy Address from the dropdown.
  2. Back in the Forethought dashboard:
    1. Paste the Identity Provider metadata in the SAML/IdP metadata URL in the input box in Step 2.
    2. Select Save.
  3. Back in the Forethought dashboard:
    1. Select Hide app (for users on desktop)
    2. Select Hide app on mobile (for users on mobile)
    3. Select Add Application and add the Bookmark App.
      4. Copy URL
    4. Enter your Forethought dashboard url in the URL input box in the Okta bookmark Add App form:
    5. (Optional) Right click and select Save Image As... the Forethought image for your Okta app:
    6. Upload the Forethought logo image in Okta to make it easy for users to recognize.
